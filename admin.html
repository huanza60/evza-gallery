Novo Catálogo

Como obter um link direto do Google Drive

  1. Faça o upload da foto no Google Drive
  2. Clique direito no ficheiro → Partilhar → Partilhar
  3. Mude para "Qualquer pessoa com o link"
  4. Clique em "Copiar link"
  5. Cole o link completo aqui — o sistema extrairá automaticamente

Formato esperado: https://drive.google.com/file/d/SEU_ID_AQUI/view?usp=sharing

Adicionar Foto ou Vídeo

Dicas para adicionar mídia

  1. Faça o upload da foto do Google Drive
  2. Selecione o catálogo de destino
  3. Cole o link do ficheiro de imagem ou vídeo
  4. Adicione uma legenda descritiva para melhor organização dos ficheiros e vídeos

Exportar Dados

Exporte todo o conteúdo da galeria como um ficheiro data.js — que pode ser carregado no repositório do GitHub para tornar as alterações permanentes.

Como tornar os dados permanentes?

  1. Clique em "Exportar Dados" — será baixado o ficheiro data.js
  2. No seu computador, substitua o ficheiro data.js atual pelo ficheiro exportado
  3. Faça o push para o repositório GitHub — o site será atualizado automaticamente pelo GitHub Pages
